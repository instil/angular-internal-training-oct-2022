In the example below, the cols change based on the size of the window.

We can use the size modifiers,

  • None (which equates to extra small)
  • sm
  • md
  • lg
  • xl

to specify certain class modifies, but only for a certain size and above.

Size Modifiers on Row Cols

This grid uses row-cols-1 row-cols-sm-2 row-cols-md-3 row-cols-lg-4 row-cols-xl-6. This means as the window size gets smaller, the number of columns decreases so each cell occupies more of the 12 columns. But the window is smaller so the columns themselves are smaller.
