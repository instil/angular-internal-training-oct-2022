Output from demos appears here

All the demos are independent of one another. They do however make use of the libraries listed below, plus the helper functions found in client-side-libraries/utils.

  • Bootstrap (CSS support)
  • Htmx (simplified AJAX)
  • Ramda (extra FP operations)
  • Monet (Monadic types)

To avoid security restrictions each demo is loaded from a locally running web server. The server can be kicked off via the start command in package.json. You will need Node.js for this to work.
